Créer avec CRA (Create-React-App).

- Lancement de l'application

```bash
npm run start
# ou
npm start
# avec Yarn
yarn start
```

- Installations des packages à la suite du clonage.

```bash
# avec NPM
npm install
# avec Yarn
yarn
```
